> MyBatis 是一款优秀的持久层框架，它支持定制化 SQL、存储过程以及高级映射。(是什么)
> MyBatis 避免了几乎所有的 JDBC 代码和手动设置参数以及获取结果集。（优势）
> MyBatis 可以使用简单的 XML 或注解来配置和映射原生信息，将接口和 Java 的 POJOs(Plain Ordinary Java Object,普通的 Java对象)映射成数据库中的记录。（怎么做的）
